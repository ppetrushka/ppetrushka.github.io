Python Scripts:

Real-Time World Population as PD parameter

  • "current world population python 3 script"

  • "current world population python 3 script (w/ Pure Data socket)"
Description:

This python script gets real time population data from worldometers.com. The first version simply prints it, where as the second version "socket" feeds it directly into pure data (or any other program for that matter). In order to run it, enable "remote automation" from your web browser settings and make sure PD is open if you're using it. Then simply open the terminal, go to the directory where the script is located and run it (>$ python3 filename.py). Make sure you're connected to the internet and have python 3 installed. If you're using PD, make sure you add a netreceive object with the argument "3000".
